# 웹 프로젝트 스터디 연습 자료

## 파일 구성
- `index.html`: 내용(문구, 구조)
- `style.css`: 꾸미기(색, 여백, 버튼, 둥근 모서리)
- `script.js`: 동작(버튼 클릭 시 이벤트)

## 연습에서 해볼 것
1. 이름 넣기
2. 만들고 싶은 사이트 한 줄 적기
3. 버튼 문구 바꾸기
4. 배경색 또는 버튼색 바꾸기
5. 버튼 클릭 시 뜨는 문구 바꾸기

## 실행 방법
- 가장 쉬운 방법: `practice/index.html` 파일을 브라우저로 열기
- 또는 VS Code에서 Live Server 확장을 써도 됨

## AI에게 물어볼 추천 프롬프트
- `index.html에서 제 이름을 OOO로 바꿔줘. 어디 줄이 바뀌는지도 설명해줘.`
- `style.css에서 버튼 색을 초록색으로 바꿔줘. 수정 코드만 보여줘.`
- `script.js에서 버튼을 누르면 '반가워요!'라고 뜨게 바꿔줘.`
